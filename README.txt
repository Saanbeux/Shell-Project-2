HOW TO COMPILE:

ECLIPSE:~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

1. Extract p1_4035_802152824_162.zip using a program such as 7zip or WinRar.
2. Open Eclipse
3. Select "File"
4. Select "Import", then "General", then "File System.
5. Select browse, then open the recently extracted folder.
6. Hit finish; project will display within Eclipse Hierarchy.
7. To run the program, select "MySystem.java", under "theSystem".

----------------------------------------------------------------------------

TERMINAL:~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

1. To compile, open the project folder from the terminal/command line. 
Write:

javac -d bin -classpath src src/theSystem/MySystem.class.

To then run, write:

java -classpath src theSystem.MySystem
